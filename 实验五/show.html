实验总结


一、document.write()、InnerHTML()和Alter
1.document.write直接输出在浏览器，后面继续write ，内容会一直在后面添加。
2.innerHTML就是获取或设置指定DOM元素(id)的内容
注意：document.write只推荐在页面onload之前使用而如果网页已经加载完毕,用innerHTML会更安全一点,用document.write会出错
3.alter的作用主要是实现弹窗的效果而本题利用重写实现效果
二、函数的使用
1.JS中的函数：把一段需要重复使用的代码，用function语法包起来，方便重复调用，分块和简化代码。复杂一点的，也会加入封装、抽象、分类等思想
2.本实验中利用许多的调用函数来实现题中的效果如Random函数等;
3.对于本次实验中的对于每个函数的调用有点击事件函数的调用，有加载网页时会自动调用等；
三、DOM(文档对象模型)的操作
1.找对象（元素节点）设置元素的属性值 设置元素的样式 动态创建和删除元素 事件的触发响应：事件源、事件、事件的驱动程序 2. var div1 = document.getElementById("box1"); //方式一：通过 id 获取 一个 元素节点（为什么是一个呢？因为 id 是唯一的）
var arr1 = document.getElementsByTagName("div"); //方式二：通过 标签名 获取 元素节点数组，所以有s
var arr2 = document.getElementsByClassName("hehe"); //方式三：通过 类名 获取 元素节点数组，所以有s
四、字符串的操作
1.字符串转为数字利用parseInt函数将字符串转为数字
2.将数字转换成字符串利用tostring函数来进行操作
